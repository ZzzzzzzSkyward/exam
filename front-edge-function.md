# 5.性能

### 离线缓存

把文件缓存到浏览器，省得浪费服务器流量。

仅查询是否有最新更新，如果有则下载并重启。

### 从各种地方获取更新

更新状态时，应当用浏览器的API通知其他页面。

打开新页面时，应当从服务器获取最新状态。

重启页面时，也应如是。

### 预读取

有必要的情况下，考试开始前预读取试题（可以使用某种加密方法，在考试开始时公布密钥）。

### 本地化

有必要的情况下，将一部分服务器的服务加载到浏览器上执行。

对于vs code，可能必须使用本地的服务器，有待研究。
